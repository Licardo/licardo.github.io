      Crontab

      • 编辑：cron -e
      • 查看任务：cron -l
      • 查看log：cat /var/log/cron.log or tail -f /var/log/cron.log
      • 重启cron服务：sudo /etc/init.d/cron restart
      • 删除crontab文件：cron -r

      注意

      第一次使用crontab会找不到cron.log文件，需要两部操作，如下：

      1. sudo vim /etc/rsyslog.d/50-default.conf 文件中的#cron.*前的#删掉
      2. sudo  service rsyslog  restart 重启log系统即可
